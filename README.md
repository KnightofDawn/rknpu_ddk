# RKNPU DDK
  RKNPU DDK is provide advanced interface to access NPU.

## Support Platform

- RK1808, RK1806
- RK1126, RK1109

Note: RK3399Pro platform is not supported.

## Support OPs
- ADD
- MULTIPLY
- CONV2D
- FULLCONNECT
- SOFTMAX
- POOL
- LEAKY_RELU
- CONCAT
- BATCH_NORM
- RESHAPE
- PERMUTE
- RELU
- SUBTRACT
- RELU6
- DATACONVERT
- SLICE
- CLIP
- GATHER
